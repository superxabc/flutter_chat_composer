# 更新日志

本项目的所有显著变更都将记录在此文件中。

格式遵循 [Keep a Changelog](https://keepachangelog.com/en/1.0.0/)，
并 adheres to [Semantic Versioning](https://semver.org/lang/zh-CN/)。

## [1.0.0] - 2025-07-18

### 新增
- Flutter 聊天组件的首次发布。
- 支持多模态输入（文字、语音、图片、文件）。
- 自适应文本输入框（2-6行）。
- 提供三种预设主题风格（扁平、简洁、自定义）。
- 支持 SVG 图标系统。
- 语音录制与手势识别功能。
- 完整的配置和回调系统。
- 无障碍功能支持。
- 独立的键盘和更多功能面板动画处理机制。
- 底部安全区域适配。
